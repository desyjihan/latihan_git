Halaman Latihan GIT

Ini halaman untuk sekedar latihan saja, jadi ngga usah terlalu serius.

git init : membuat repo git git status : mengecek status git git add -A : menambahkan file ke temp git commit : menyimpan file di repo lokal git push : mengirim perubahan ke server git pull : download perubahan dari server git clone : download repo baru dari server
